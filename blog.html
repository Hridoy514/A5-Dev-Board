What are the different ways to select an element in the DOM?

There are several ways to select elements from the page. The easiest and fastest way is getElementById(), which finds an element using its unique id. If you need to select multiple elements that have the same class name, you can use getElementsByClassName(), which returns a list of all elements with that class. Similarly, getElementsByTagName() selects all elements of a certain type, like all p tags or all div tags.A more flexible way is querySelector(), which allows you to use CSS-style selectors to find elements. For example, document.querySelector(".button") finds the first element with the class button, while querySelectorAll(".button")selects all elements with that class.

What is the difference between innerHTML, innerText, and textContent ?

When working with text inside an element on a webpage, you can use innerHTML, innerText, or textContent, but they work differently. innerHTML gets or sets everything inside an element, including HTML tags. innerText gets or sets only the visible text inside an element. If an element has display: none, innerText will not show that text. textContent is similar to innerText, but it keeps all text, including hidden text. Unlike innerHTML, it treats everything as plain text that's why use innerHTML if you need to work with HTML tags, use innerText for visible text, and use textContent to get all text.

What is event delegation in the DOM?

Event delegation is a smart way to handle events in JavaScript, especially when working with many elements. Event delegation makes your code faster, cleaner, and more efficient, especially when new elements are added dynamically. Event delegation helps reduce the number of event listeners, saving memory and improving performance on your website.

What is event bubbling in the DOM?

Event bubbling is how an event moves from the bottom to the top in the webpage. When you click on a small element inside a bigger element, the event starts from the clicked element and then moves up to its parent, then to the grandparent, and so on until it reaches the top html element. How do you create, add, and remove elements using JavaScript?

In JavaScript, I can create, add, and remove elements to change a webpage. To create an element, I use document.createElement("tagName"). For example, document.createElement("p") makes a new p tag.Then I can do this using appendChild() to add it at the end of a parent element or prepend() to add it at the beginning. For example, parent.appendChild(newElement) puts the new element inside the parent. If I want to remove an element, I use remove() or removeChild(). For example, element.remove() deletes an element directly, or parent.removeChild(childElement) removes a child from its parent.
